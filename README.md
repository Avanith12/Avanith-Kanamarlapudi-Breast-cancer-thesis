# mpsych.org

**Get Ruby >=2.7 <3.0**<br />
sudo apt-get install ruby-bundler ruby-dev<br />

gem install github-pages -v '198' --source 'https://rubygems.org/'<br />

bundle install<br />

**Run the server**<br />
bundle exec jekyll serve<br />

**Connect to localhost**<br />
http://127.0.0.1:4000/<br />
